post /authenticate

Authenticate user

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • email: required (string)

    Needs to be valid email address

  • password: required (string - minLength: 8)

    Password needs to have:

    • One uppercase character
    • One lowercase character
    • One number

Examples:

{
  "email": "regular.user@mail.com",
  "password": "Password123"
}

HTTP status code 200

Successfully authenticated user.

HTTP status code 400

Bad requests made for authentication. There are two scenarios.

  • Wrong parameters in body
  • Wrong password

Body

Media type: application/json

Type: object

Properties
  • message: required (string)

    Error message thrown from API.

  • debugInfo: (array)

    Array of objects containing path to wrong property and error message.

  • error: required (object)

Examples:

{
  "message": "Parameters error",
  "debugInfo": [
    {
      "path": "email",
      "message": "Email is required"
    },
    {
      "path": "password",
      "message": "Password is required"
    }
  ],
  "error": {
    "message": "Parameters error",
    "debugInfo": [
      {
        "path": "email",
        "message": "Email is required"
      },
      {
        "path": "password",
        "message": "Password is required"
      }
    ],
    "status": 400
  }
}
{
  "message": "Wrong password",
  "error": {
    "message": "Wrong password",
    "status": 400
  }
}
